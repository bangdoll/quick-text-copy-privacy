Quick Text Copy 隱私政策

最後更新日期：2025年8月1日

概述

Quick Text Copy 是一個簡單的 Chrome 瀏覽器擴充功能，旨在幫助使用者快速複製當前網頁的標題和網址到剪貼簿。我們非常重視您的隱私，本政策說明我們如何處理您的資訊。

資料收集聲明

我們不收集任何個人資料。

我們不收集的資料：

  • ❌ 個人身份資訊（姓名、電子郵件、電話號碼等）
  • ❌ 瀏覽歷史記錄
  • ❌ 網站內容或使用者輸入
  • ❌ 位置資訊
  • ❌ 裝置資訊
  • ❌ 使用統計資料
  • ❌ 任何形式的追蹤資料

我們不進行的活動：

  • ❌ 不向第三方傳送任何資料
  • ❌ 不使用分析工具或追蹤服務
  • ❌ 不儲存任何使用者資料到本地或雲端
  • ❌ 不建立使用者檔案或行為模式
  • ❌ 不投放廣告或進行商業追蹤

擴充功能運作方式

Quick Text Copy 的運作完全在您的本地裝置上進行：

  1. 本地處理：當您點擊擴充功能圖示時，它會讀取當前分頁的標題和網址
  2. 格式化：將標題和網址組合成 "標題 網址" 的格式
  3. 複製到剪貼簿：將格式化的文字複製到您的系統剪貼簿
  4. 無資料傳輸：整個過程不涉及任何網路傳輸或資料儲存

權限說明

我們的擴充功能僅請求以下最小必要權限：

activeTab 權限

  • 用途：讀取當前活躍分頁的標題和網址
  • 範圍：僅限當前分頁，且僅在使用者主動點擊擴充功能時
  • 資料處理：讀取的資料僅用於格式化和複製，不會儲存或傳輸

scripting 權限

  • 用途：在當前分頁中執行複製到剪貼簿的腳本
  • 範圍：僅執行複製功能，不修改頁面內容
  • 資料處理：不讀取或修改頁面上的任何其他內容

資料安全

雖然我們不收集任何資料，但我們仍然採取以下安全措施：

  • ✅ 使用 Manifest V3 最新安全標準
  • ✅ 實施嚴格的內容安全政策 (CSP)
  • ✅ 最小權限原則，僅請求必要權限
  • ✅ 本地處理，無網路通訊
  • ✅ 開源透明，程式碼可供檢視

第三方服務

我們不使用任何第三方服務，包括但不限於：

  • 分析服務（如 Google Analytics）
  • 廣告網路
  • 社交媒體整合
  • 雲端儲存服務
  • 外部 API 或服務

兒童隱私

由於我們不收集任何個人資料，本擴充功能適合所有年齡層使用，包括 13 歲以下的兒童。我們不會故意收集兒童的個人資訊。

政策變更

如果我們的隱私政策有任何變更，我們會：

  1. 更新本文件的「最後更新日期」
  2. 在擴充功能更新說明中註明重要變更
  3. 如有重大變更，會在 Chrome Web Store 頁面上說明

聯絡我們

如果您對本隱私政策有任何疑問或疑慮，請透過以下方式聯絡我們：

  • GitHub Issues：專案頁面
  • 電子郵件：your-email@example.com

法律依據

本隱私政策符合以下法規要求：

  • 歐盟一般資料保護規則 (GDPR)
  • 加州消費者隱私法 (CCPA)
  • Chrome Web Store 開發者政策
  • Google 隱私要求

使用者權利

由於我們不收集任何個人資料，以下權利自動得到保障：

  • ✅ 資料可攜權：無資料需要匯出
  • ✅ 刪除權：無資料需要刪除
  • ✅ 更正權：無資料需要更正
  • ✅ 存取權：無個人資料被處理
  • ✅ 反對權：無資料處理可反對

簡而言之

Quick Text Copy 是一個完全本地運作的工具，不收集、不儲存、不傳輸任何使用者資料。您的隱私得到完全保護。
